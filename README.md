# Travelgram (Responsivo)

Projeto de página estática responsiva para um perfil de viagens, com seções de perfil, galeria e rodapé.

## Tecnologias

- HTML5
- CSS3 (layout com Grid/Flex e utilitários)

## Como visualizar

1. Abra o arquivo index.html no navegador.
2. Ajuste a largura da janela para ver o comportamento responsivo.

## Estrutura de pastas

- index.html: marcação principal
- styles/: estilos do projeto
  - global.css
  - utility.css
  - header.css
  - about.css
  - gallery.css
  - footer.css
- assets/: imagens e ícones

## Observações

- O layout utiliza a classe container para limitar largura em telas grandes.
- A seção #about tem fundo aplicado via CSS.
